# TalkBoard
Forum management platform
## installation
```
$ npm install
```
## Running the app
*We are split two folder: frontend and backend*

**frontend**
```bash
#development

$ npm run dev

# build mode

$ npm run build
```
**backend**
```bash
#development

$ npm run start

#watch mode

$ npm run start:dev

#production mode

$ npm run start:prod
```
